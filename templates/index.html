About This Project

This project is a Machine Learning-based Student Marks Predictor designed to estimate a student's exam score using academic and lifestyle-related factors such as study hours, attendance, sleep quality, study method, facility rating, and more.

The complete machine learning workflow was implemented, including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, hyperparameter tuning using GridSearchCV, model evaluation, and deployment through a Flask web application.
