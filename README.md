# Student Score Predictor

A machine learning web application that predicts a student's exam score based on input features like study hours, internet access, study methods, sleep hours, gender etc. The model is deployed using Flask for real-time predictions.

---

## Live Demo
[Click here to view project](https://students-marks-predictor-1.onrender.com/) It will take few seconds to open because i am using free tier of render to deploy the model.

---

## Project Overview
This project is built to predict student exam scores using a regression model. It helps understand how different academic factors affect final score.

---

## Features
- Predicts student exam score based on inputs
- Simple web interface
- Real-time prediction using ML model
- Data preprocessing included in pipeline
- Flask based deployment

---

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- HTML, CSS

---

## Machine Learning Model
- Model: Linear Regression / Decision Tree / Random Forest / Gradient Boosting/ XGBoost
- Dataset: Student performance dataset
- Metrics: MAE, RMSE, R² score
